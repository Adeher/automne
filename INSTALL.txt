1 - Prérequis
-------------------------------------------------------

Système Linux, Windows (NT, 2000, XP ou 2003) ou Solaris.
PHP 5.2.0 minimum (pour des raisons de sécurité, la dernière version 5.x est conseillée).
Apache 1.3.x ou 2.x.x (pour des raisons de sécurité, la dernière version 1.3.xx est conseillée).
MySQL 5.0.x (pour des raisons de sécurité, la dernière version 5.0.x est conseillée).

Pour de meilleures performances, installez PHP avec le module CLI (Command Line Interface) voir www.php.net pour plus d'informations.


2- Installation
-------------------------------------------------------

Copiez les fichiers install.php et automne-xxx.tar.gz contenu dans l'archive téléchargée sur Sourceforge à la racine de votre site web.
Attention, il est impératif qu'Automne soit installé à la racine de votre serveur web.
Installer Automne dans un sous repertoire de cette racine ne fonctionnera pas.
L'adresse par laquelle Automne sera accessible doit être de la forme :
http://domaine.tld/
et non :
http://domaine.tld/repertoire/

Lancez le fichier install.php depuis un navigateur web (l'adresse doit donc être http://domaine.tld/install.php)
Suivez ensuite les instructions données par le script d'installation.

Notes : 
- Si vous avez installé le module CLI de PHP, choisissez durant l'installation l'option "Script en tache de fond".
- Si vous installez Automne sur un serveur Windows, veillez à ne pas avoir d'espace dans le chemin d'installation.
- Le fichier atm-test-conf.php vous permet de tester la validité de la configuration de votre serveur avant d'installer Automne.

/!\ Important
-------------------------------------------------------
Une fois l'installation complétée, supprimez les fichiers install.php ainsi que automne-xxx.tar.gz
Laisser ces fichiers sur un site en production représente une faille de sécurité très importante !

Vous trouverez toute l'aide nécessaire pour l'utilisation du logiciel sur http://www.automne.ws et sur http://doc.automne.ws

ENGLISH VERSION
-------------------------------------------------------

1 - System requirements
-------------------------------------------------------

Linux, Windows (NT, 2000, XP or 2003) or Solaris system.
PHP 5.2.0 (for security matters, we recommend the last version 5.x).
Apache 1.3.x or 2.0.x (for security matters, we recommend the last version 1.3.xx).
MySQL 5.0.x (for security matters, we recommend the last version 5.0.x).

For greater performance, install PHP with the CLI (Command Line Interface) module : see www.php.net for more information.


2- Installation
-------------------------------------------------------

Copy the content from the install.php and automne-xxx.tar.gz files in the archive downloaded from Sourceforge to the root of your website. Caution : Automne has to be installed on the root of your web server. If you install Automne in a sub-directory of this root, it will not function.
The address where Automne will be reachable must be of this type :
http://domain.tld/
and not :
http://domain.tld/directory/

Launch the install.php file from a web browser (the address will be : http://domain.tld/install.php)
Then follow the instructions given by the installation script.

NB : 
- If you installed the CLI module of PHP, choose the "Background script" option during the installation.
- If you install Automne on a Windows server, take care to not have spaces in the installation path (example:  by default, easyphp use the folder program files what will not function with Automne).
- The file atm-test-conf.php allows you to test the validity of the server configuration before installing Automne.
/!\ Important
-------------------------------------------------------
Once you completed the installation, delete the following files : install.php and automne-xxx.tar.gz
Keeping this files on a website on production may represent a security threat!

You will find all necessary assistance for the use of the Software on http://www.automne.ws and http://doc.automne.ws
