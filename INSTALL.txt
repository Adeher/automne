1 - Prérequis
-------------------------------------------------------

Système Linux, Windows (NT, 2000, XP ou 2003) ou Solaris.
PHP 4.3.8 minimum (pour des raisons de sécurité, la version 4.4.2 est conseillée. Automne n'est pas compatible avec PHP 5.x pour le moment).
Apache 1.3.x ou 2.0.x (pour des raisons de sécurité, la version 1.3.34 est conseillée).
MySQL 4.0.x (une version 3.23.xx fonctionnera mais les futures versions d'Automne seront incompatibles avec cette version de MySQL).

Pour de meilleures performances, installez PHP avec le module CLI (Command Line Interface) voir www.php.net pour plus d'informations.


2- Installation
-------------------------------------------------------

Copiez les fichiers install.php et automne-xxx.tar.gz contenu dans l'archive téléchargée sur Sourceforge à la racine de votre site web.
Attention, il est impératif qu'Automne soit installé à la racine de votre serveur web.
Installer Automne dans un sous repertoire de cette racine ne fonctionnera pas.
L'adresse par laquelle Automne sera accessible doit être de la forme :
http://sous_domaine.domaine/
et non :
http://sous_domaine.domaine/repertoire/

Lancez le fichier install.php depuis un navigateur web (l'adresse doit donc être http://sous_domaine.domaine/install.php)
Suivez ensuite les instructions données par le script d'installation.

Notes : 
- Si vous avez installé le module CLI de PHP, choisissez durant l'installation l'option "Script en tache de fond". Vous devez pour cela connaitre l'emplacement de ce module CLI sur le serveur.
- Si vous installez Automne sur un serveur Windows, veillez à ne pas avoir d'espace dans le chemin d'installation (exemple : par défaut, easyphp s'installe dans le répertoire program files ce qui ne fonctionnera pas avec Automne).

/!\ Important
-------------------------------------------------------
Une fois l'installation complétée, supprimez les fichiers install.php ainsi que automne-xxx.tar.gz
Laisser ces fichiers sur un site en production représente une faille de sécurité très importante !

Un ensemble de documents techniques utiles sur Automne sont disponibles en téléchargement sur Sourceforge.
Des guides d'administration courant d'Automne sont disponibles dans l'administration du logiciel (voir l'icone "?" en haut à droite de l'interface d'administration).


-------------------------------------------------------
ENGLISH VERSION
-------------------------------------------------------

1 - System requirements
-------------------------------------------------------

Linux, Windows (NT, 2000, XP or 2003) or Solaris system.
PHP 4.3.8 (for security matters, we recommend version 4.4.2. Automne is not compatible with PHP 5.x for the moment).
Apache 1.3.x or 2.0.x (for security matters, we recommend version 1.3.34).
MySQL 3.23.x (for evolution matters, we recommend version 4.0.x).

For greater performance, install PHP with the CLI (Command Line Interface) module : see www.php.net for more information.


2- Installation
-------------------------------------------------------

Copy the content from the install.php and automne-xxx.tar.gz files in the archive downloaded from Sourceforge to the root of your website. Caution : Automne has to be installed on the root of your web server. If you install Automne in a sub-directory of this root, it will not function.
The address where Automne will be reachable must be of this type :
http://sub_domain.domain/
and not :
http://sub_domain.domain/directory/

Launch the install.php file from a web browser (the address will be : http://sub_domain.domain/install.php)
Then follow the instructions given by the installation script.

NB : 
- If you installed the CLI module of PHP, choose the "Background script" option during the installation. To do so, you will need to know the location of this module on the server.
- If you install Automne on a Windows server, take care to not have spaces in the installation path (example:  by default, easyphp use the folder program files what will not function with Automne).

/!\ Important
-------------------------------------------------------
Once you completed the installation, delete the following files : install.php and automne-xxx.tar.gz
Keeping this files on a website on production may represent a security threat!

Several useful technical documents about Automne can be downloaded on Sourceforge.
Administration guides are also available in Automne's administration (click on the "?" icon in the top right corner of the administration platform).
