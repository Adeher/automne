Installation : pas à pas
------------------------

Cette version fourni tout ce dont vous avez besoin pour faire fonctionner Automne 4 sur Windows. 
Attention, cette version n'est pas sécurisée. Elle ne convient qu'à un travail local et ne doit pas être employée sur un serveur de production. Pour plus d'informations à ce sujet, une fois l'installation terminée, allez à l'adresse http://127.0.0.1/xampp/

- Décompressez l'archive sur l'un de vos disque (directement dans C: par exemple). Cela créera un répertoire xampplite.
- Lancez le fichier xampplite\setup_xampp.bat pour configurer automatiquement les chemins de l'application
- Lancez le fichier xampplite\xampp-control.exe pour démarrer les serveurs Apache et MySQL
- Sur l'écran qui s'affiche, démarrez à l'aide des boutons "Start" Apache puis MySQL
- Vous accédez à l'administration d'Automne à l'adresse : http://127.0.0.1/automne/admin/ (login : root / automne)
- Par défaut, le site est installé avec la Démo anglaise d'Automne. Pour Installer la Démo française d'Automne ou une version vide, relancez l'installeur à l'adresse http://127.0.0.1/install.php et lors de la sélection du type d'installation, choisissez le type d'installation que vous souhaitez.

Vous trouverez les utilitaires de configuration de XAMPP à l'adresse http://127.0.0.1/xampp/
Vous trouverez les fichiers d'Automne dans le répertoire xampplite\htdocs

Si vous rencontrez un problème ou si vous avez besoin d'aide, vous pouvez à tout moment nous poser vos questions sur le forum ou nous envoyer un mail à contact[at]automne.ws avec le fichier de diagnostic généré par Automne.

Mettre à jour Automne
---------------------

Pour mettre à jour votre Automne, il faut passer successivement les patchs de mises à jour téléchargeables sur http://www.automne.ws/download/

Configuration serveur
---------------------

 	- PHP version 5.2 ou supérieur
 	- MySQL version 5.0.x ou supérieur
 	- Apache 1.3.x ou supérieur

Pour de meilleures performances, installez PHP avec le module CLI (Command Line Interface) voir http://php.net/manual/features.commandline.php pour plus d'informations.
